# Weather App (PyQt5)

A desktop weather application built with Python and PyQt5 that retrieves real-time weather data using the OpenWeatherMap API.

## Features

- Search weather by city name
- Real-time weather data
- Displays temperature in Celsius
- Weather description
- Weather emoji based on weather condition
- Error handling for network and API requests

## Technologies Used

- Python
- PyQt5
- Requests
- OpenWeatherMap API

## Installation

```bash
git clone https://github.com/daiyasser/WeatherApp-PyQt5-.git
cd WeatherApp-PyQt5-
pip install -r requirements.txt
python main.py
```
